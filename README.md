# Cloud Automation Basics

Repositório dedicado a estudos e exemplos básicos de automação aplicada a ambientes de Cloud Computing.

O foco está na compreensão de conceitos de automação, infraestrutura como código e padronização de ambientes, sem complexidade excessiva.

## Conteúdos
- Introdução à automação em Cloud
- Scripts básicos (exemplos simples)
- Conceitos de Infraestrutura como Código
- Organização e padronização de ambientes
- Boas práticas iniciais

## Objetivo
Desenvolver entendimento prático sobre automação em Cloud, conectando conceitos técnicos à realidade corporativa e educacional.

Projeto criado para estudo contínuo e evolução técnica.
